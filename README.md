# AI-Powered Real-Time Speech Translation

This project is an AI-powered application that allows real-time translation of speech in videos. It supports multiple languages and provides an interactive interface for uploading videos, extracting audio, translating, and generating output.

## Features

- Real-time speech translation from video/audio.
- Supports multiple languages.
- Translated subtitles can be added to videos.
- Microphone input feature is available **only on the local version** and does **not work in the deployed version**.

## Deployment

You can try the deployed version of the app here:  
[https://ai-powered-real-time-speech-translation.streamlit.app/](https://ai-powered-real-time-speech-translation.streamlit.app/)

> ⚠️ **Note:** Microphone input is currently functional only when running the app locally. The deployed version does not support this feature due to browser and server restrictions.
